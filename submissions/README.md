# submissions/

候选人答案提交在这里，每人一个目录：

```
submissions/
└── <你的GitHubID>/
    ├── solution.md      ← 你的答案（从 template/solution.md 复制）
    └── assets/          ← 图片等资源（可选）
```

## 命名规范

- **目录名**：你的 GitHub ID（区分大小写，与 fork 账号一致）。
- **主文件**：统一命名为 `solution.md`。
- **图片**：放在同目录 `assets/` 下，用相对路径引用，例如 `![](./assets/loop.png)`。

## 关于公开性

题库开源，你提交的 PR 对所有人可见。详见招聘主站 [笔面试流程](http://211.159.165.135:12345/assessment)。
